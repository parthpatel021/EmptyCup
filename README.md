## Setting up Locally
1. Clone this repository to your local environment

```bash
git clone https://github.com/parthpatel021/EmptyCup.git
```

2. Install dependencies of client

```bash
cd ./client/
npm install
```

3. To run it locally execute following command in `client` directory

```bash
npm start
```
